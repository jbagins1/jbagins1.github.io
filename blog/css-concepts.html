Lets talk about some important CSS concepts: margin, border, and padding. These concepts are important to understand because when discussing CSS design and layout, the "box model" is used. The "box model" is essentially a box that consist of the content, the padding, the border, and the margin. These concepts make up what is known as the "box model" and allow us to define space between elements. A visual representation of these concepts is shown in the diagram below.

The padding is a transparent area around the content. Padding is the horizontal and vertical space thats set around the content area of the element inside of the box. Padding separates the content of the element from the element's edges. Padding can be set either by using the shorthand (padding) or the longhand (padding-top, padding-right, padding-bottom, and padding left). On the other hand, the border goes around the padding and the content. Meanwhile, the margin is a transparent area outside of the border. If elements have no margins around them, generally, they will appear right next to each other. Margins can be set using either the shorthand (top, right, bottom, and left) or the longhand (margin-top, margin-right, margin-bottom, and margin-left).
In sum, essentially padding adds space inside of an element, separating the elements content from the edges of the targeted element. The border encapsulates the padding and the content. The margins separate page element's from each other and away from page edges.
