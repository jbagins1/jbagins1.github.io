Enumerable Method: Map Method - February 4, 2015

Returns a new array with the results of running block once for every element in enum. Example: (1..4).collect { |x| x*x } => [1, 4, 9, 16] If no block is given an enumerator is returned. Example: (1..4).collect { "cat" } => ["cat", "cat", "cat", "cat"]
